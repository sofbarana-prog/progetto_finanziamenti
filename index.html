Questionario Progetto — Funding Call Matcher

Questo strumento suggerisce call coerenti con il profilo del progetto. Il filtro sul tipo di azione viene applicato solo alle call di Horizon Europe; gli altri programmi restano inclusi se coerenti con l'area tematica e con la timeline del progetto.

1) Timeframe di chiusura del progetto
2) Natura del progetto
3) Cluster tematico
4) Tipo di azione
5) Beneficiario di altri schemi
6) Tipologia partner
Il tool usa calls.json. Per pubblicarlo online è sufficiente caricare questo file HTML e il JSON su GitHub Pages, Netlify o Vercel.
